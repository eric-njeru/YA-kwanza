The alt attribute is typically a description of the image, and is optional. However, it's a good idea to include for a few reasons. First, if the image fails to load, the user can hover the mouse over the image's area and read the alt text for a description of the image. Second, visually impaired users often browse the web with the aid of screen reading software. When you include the alt attribute, the screen reading software can read the image's description out loud to the user. Typically, websites will store their images in a separate folder from the rest of the HTML. Create a folder named images in the main project directory and move the image there.
